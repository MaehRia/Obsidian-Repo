# Урок 4. (02.02)

### Падежи


1. Именительный падеж (Кто? Что?): ребёнок, шапка.
2. Родительный падеж (Кого? Чего?): нет ребёнка, шапки.
3. Дательный падеж (Кому? Чему?): дам ребёнку, рад каникулам
4. Винительный падеж (Кого? Что?): вижу ребёнка, шапку.
5. Творительный падеж (Кем? Чем?): доволен ребёнком, шапкой.
6. Предложный падеж (О ком? О чём?): думаю о ребёнке, шапке.

| ------------ | --------------------- |
| Именительный | есть                  |
| Родительный  | нет                   |
| Дательный    | рад, дать             |
| Винительный  | вижу, винить          |
| Творительный | любуюсь               |
